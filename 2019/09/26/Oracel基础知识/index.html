查询语句结构

select [列名] [*] from 表名 [where条件] [ group by 分组条件] [having 过滤] [order by 排序]

伪表dual

dual：oracle中的虚拟表，伪表，主要是用来补齐语法结构；

select 1+1 from dual;

注意：直接写一个常量比*效率高

select count(1) from emp;

select count(*) from emp;

别名查询

使用as关键字，可以省略，别名中不能有特殊字符或者关键字，如果有就加双引号（比如空格）

select name as 名称 from emp;

去除重复数据 distinct

加在select和列名后面，多列去除重复；每一列都一样才能够算作是重复

select distinct job from emp; – 单列去除重复

查询中四则运算

select 1+1 from dual;

select id*12 from emp;

Oracle中，双引号主要是别名的时候使用，单引号是使用的值是字符.

空值处理

nvl(参数一，参数二)：如果1为null,则返回2，不为null就为1；

nvl2(参数一，参数二，参数三)：1为null,则返回3，不为null，则返回2；

字符串拼接

Oracle特有的连接符：|| 拼接

concat(str1,str2)函数

select ‘姓名：’ || name from emp;

查询条件

​ 条件查询就是where后面的写法

关系运算符和排序

=、 <、 >、 >=、 <=、 !=、 <>

注意：<>效率比!=

逻辑运算符

and or not

其他运算符

like 模糊查询

in(set) 在某个集合内

between..and.. 在某个区间内

is null 判断为空

is not null 判断不为空

模糊查询 like

%匹配多个字符

_匹配单个字符

如果有特殊字符使用escape转义

select * from emp where ename like ‘%%%’ escape ‘\‘;

排序 order by

升序：asc ascend

降序：desc descend

注意：排序注意null问题：nulls first | last指定null值显示位置

同时排列多列，用逗号隔开
